# Shri Radhe Krishan Mehandi Training Centre

Animated single-page website for a bridal mehandi artist and training centre in Birgunj, Nepal —
13+ years of artistry, 7,000+ students trained since 2011.

**Live site:** https://hesignberg-0.github.io/shri-radhe-krishan-mehandi/

## Highlights

- SVG intro animation: a hand outline draws itself, then a full mehandi design blooms onto it stroke by stroke
- Hero with floating real-work photos, rotating mandala, and parallax paisley motifs
- Services, training journey, admission form (submits as a pre-filled WhatsApp message — no backend needed)
- Founder section, real-work gallery, FAQ accordion, live Google Map, WhatsApp/Instagram/TikTok contact cards
- Pure HTML/CSS/JS — no build step, no dependencies beyond Google Fonts

## Local preview

```powershell
powershell -NoProfile -File serve.ps1 -Port 8145
# then open http://localhost:8145/
```

Or just open `index.html` in a browser.

## Deploy

Hosted on GitHub Pages from the `main` branch root — every push to `main` redeploys automatically.
